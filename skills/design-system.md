# FQ Design System Skill

Read this before making ANY UI changes, adding 
new components, or styling anything in the app.

---

## The Golden Rule

Never use raw Tailwind colors (blue-500, gray-200 etc).
Always use FQ design tokens. The app has a specific 
aesthetic — warm, minimal, luxury. Every component 
should feel consistent.

---

## Color Tokens

| Token | Usage |
|-------|-------|
| `fq-dark` | Primary text, buttons, headings |
| `fq-muted` | Secondary text, labels, placeholders |
| `fq-bg` | Page background (warm off-white) |
| `fq-card` | Card/panel background (slightly lighter) |
| `fq-border` | All borders and dividers |
| `fq-accent` | Brand accent — use sparingly |
